# styler

This is a collection of scripts I use in the chrome [styler](https://chrome.google.com/webstore/detail/styler/bogdgcfoocbajfkjjolkmcdcnnellpkb) plugin.

### YouTube

This adds a link to the embedded video so you can watch it full window (not full screen). So this: `https://www.youtube.com/watch?v=oEt08GK7oeA` becomes this: `https://www.youtube.com/embed/oEt08GK7oeA?autoplay=1`. The link text is a `â˜¯` symbol near the title.
